数据库（Mysql）

  • 连接Mysql：mysql -h localhost -P 3306 -u root -p密码（登陆前确保服务已启动）
  • 启动服务：net start mysql
  • 关闭服务：net stop mysql

数据库的三层结构

  • 安装Mysql数据库就是在主机安装一个数据库管理系统（DBMS），这个程序可以管理多个数据库。DBMS: database manage system
  • 一个数据库中可以创建多个表，以保存数据（信息）
  • DBMS、数据库、表的关系

创建数据库

  • CHARACTER SET：指定数据库采用的字符集
  • COLLATE：指定数据库字符集的校对规则
    • utf8_bin ：区分大小写
    • utf8_general_ci : 不区分大小写（默认是这个）

SQL语句分类

  • DDL：数据定义语句【creat 表，库…】
  • DML ：数据操作语句【增加 insert ， 修改 updata ， 删除 delete】
  • DQL ：数据查询语句【select 】
  • DCL：数据控制语句【管理数据库：比如用户权限 grant（授权）、revoke（撤销授权）】

数据库的操作

  • 创建数据库：CREAT DATABASE asahi_01 CHARACTER SET utf8 COLLATE utf8_bin;

  • 删除数据库：DROP DATABASE asahi_01;

  • 查询一条sql：

  • 显示数据库语句：SHOW DATABASES;

  • 显示数据库创建语句：SHOW CREAT DATABASE asahi_01

  • 备份数据库 ：mysqldump -u root -p -B database01 database02 > d: \\bak.sql【在DOS下执行】

  • 恢复数据库：sourse d:\\bak.sql【进入mysql命令行执行】

使用细节

数据库的CRUD语句

  • 增：INSERT
  • 删：DELETE
  • 改：UPDATE
  • 查：SELECT

INSERT语句

  • 按列顺序可以不写列名
  • 可以单独添加列的值（前提是其他可以为null或有默认值）

UPDATE语句

  • SET:指示要修改哪些列和要给予那些值
  • WHERE：指定要修改哪些行。【如果没有WHERE子句，则更新所有】

追加/删除语句

DELETE语句

  • 如果没有WHERE【删除表中所有数据】
  • DELETE只删除表的数据，不删除表本身
  • 不能单独删去某个字段，可以用UPDATE置空

统计函数

  • count() : 返回行的总数
    • count(*) ：返回所有行数
    • count(列名) ：返回所有非null行数
  • sum()：返回满足where条件的行数【用于列为数值】
  • avg()：返回满足where条件的平均值【用于列为数值】
  • max()/min()：统计最大/最小值
  • group by（分组）、having（过滤）
